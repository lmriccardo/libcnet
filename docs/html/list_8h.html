size_t _vsize )
extern

Construct and return the pointer to a new Node initialized with given input values. Notice that, this function dynamically allocate memory when creating the new Node. Hence, the node should be freed using the Node_delete function. If the node is in a Linked List, then all nodes will be freed when the Linked List is deleted.
