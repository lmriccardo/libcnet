\doxysubsection{Detailed Description}
Struct representing a \doxyref{Sender}{p.}{structSender}. It is called Raw since it uses the SOCK\+\_\+\+RAW option when creating a new socket.

This structure has the following fields\+:


\begin{DoxyItemize}
\item {\ttfamily \+\_\+srcaddress} The IP address sending the packets
\item {\ttfamily \+\_\+dstaddress} The host receiving the packet
\item {\ttfamily \+\_\+gateway} The gateway address of the current Source address
\item {\ttfamily \+\_\+socket} The file descriptor of the created socket
\item {\ttfamily \+\_\+msgcnt} The current Message number being sent
\item {\ttfamily \+\_\+proto} The protocol used to send the packets
\item {\ttfamily \+\_\+lastid} The last used IP Packet Identifier
\item {\ttfamily \+\_\+lasticmpid} The last used ICMP Packet Identifier
\item {\ttfamily \+\_\+icmpsn} The last ICMP Message Sequence Number
\item {\ttfamily \+\_\+timer} A \doxyref{Timer}{p.}{structTimer} shared with the \doxyref{Receiver}{p.}{structReceiver}
\item {\ttfamily \+\_\+mtu} The Maximum Transmission Unit
\item {\ttfamily \+\_\+mutex} A simple Mutex to syncrhonize with the receiver
\item {\ttfamily \+\_\+synch} A boolean flag to indicate the syncrhonization
\item {\ttfamily \+\_\+sent} A boolean flag to help avoid deadlock when synchronizing
\item {\ttfamily \+\_\+params} Some additional parameters for IP Packets
