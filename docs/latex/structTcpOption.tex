\doxysubsection{Detailed Description}
Struct representing a single generic TCP Option. Since options can be of variable size we need to use {\ttfamily void} pointers to address generic type values. Notice that the field {\ttfamily \+\_\+value} MUST contain the entire option value. It means that, if the option have two 2 bytes values, we can write it into the {\ttfamily \+\_\+value} field has {\ttfamily (val1 $<$$<$ 16) + val2} or as a vector of u\+\_\+int16\+\_\+t.
