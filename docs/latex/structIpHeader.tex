\doxysubsection{Detailed Description}
Struct representing the IP Header of the IP Packet. It consists of classical 28 bytes divided into\+:


\begin{DoxyItemize}
\item {\ttfamily \+\_\+version}, IP Protocol Version + IHL (8 bits)
\item {\ttfamily \+\_\+dsf}, Type of service (8 bits)
\item {\ttfamily \+\_\+tlength}, Total length (16 bits)
\item {\ttfamily \+\_\+id}, Identification (16 bits)
\item {\ttfamily \+\_\+flag\+\_\+off}, Flags and Fragment Offset (16 bits)
\item {\ttfamily \+\_\+ttl}, Time to Live (8 bits)
\item {\ttfamily \+\_\+protocol}, IP Protocol (8 bits)
\item {\ttfamily \+\_\+hdr\+\_\+chksum}, Checksum (16 bits)
\item {\ttfamily \+\_\+srcaddr}, Source Address (32 bits)
\item {\ttfamily \+\_\+dstaddr}, Destination Address (32 bits).
\end{DoxyItemize}

This structure refers to the RFC 791 {\texttt{ https\+://datatracker.\+ietf.\+org/doc/html/rfc791}} The TOS field has been subtituted with the Differentiated Service Field according to the RFC 2474 {\texttt{ https\+://datatracker.\+ietf.\+org/doc/html/rfc2474}}
