\doxysubsection{Detailed Description}
Struct representing the TCP header of the TCP Packet. It consists of classical minimum of 20 bytes divided into\+:


\begin{DoxyItemize}
\item {\ttfamily \+\_\+srcport}, source port (2 bytes)
\item {\ttfamily \+\_\+dstport}, destination port (2 bytes),
\item {\ttfamily \+\_\+seqnum}, sequence number (4 bytes)
\item {\ttfamily \+\_\+acknum}, acknowledgment number (4 bytes)
\item {\ttfamily \+\_\+offset}, Data Offset + reserved bits (1 byte)
\item {\ttfamily \+\_\+flags}, Control bits or flags (1 byte)
\item {\ttfamily \+\_\+window}, \doxyref{Receiver}{p.}{structReceiver} Window size (2 bytes),
\item {\ttfamily \+\_\+checksum}, Checksum (2 bytes)
\item {\ttfamily \+\_\+urgpntr}, Urgent Pointer (2 bytes)
\item {\ttfamily \+\_\+options}, Optional Options (from 0 to 20 bytes).
\end{DoxyItemize}
