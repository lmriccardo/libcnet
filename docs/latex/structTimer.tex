\doxysubsection{Detailed Description}
Struct representing a simple timer. It can be used to time the elapsed time between the message being sent and the response being received by the receiver. The timer starts by calling the corresponding {\ttfamily Timer\+\_\+start} function.

The means of the fields is the following\+:
\begin{DoxyItemize}
\item {\ttfamily \+\_\+start} the time when the \doxyref{Timer}{p.}{structTimer} is created
\item {\ttfamily \+\_\+elapsed} the elapsed time between start and current
\item {\ttfamily \+\_\+current} the current time everytime updated
\item {\ttfamily \+\_\+previous} the time when the {\ttfamily Timer\+\_\+get\+Time} function is called 
\end{DoxyItemize}
